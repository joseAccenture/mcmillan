Llista de permisos

Asignar rol

Permite asignar y revocar un rol a un usuario.

Autorizar circuito

Permite autoriza a un par back office / organismo para ejecutar sus solicitudes sobre un circuito concreto. Incluye tanto la concesión como la revocación de esta autorización.

Consulta

Permite la consulta (no modificación) de peticiones, solicitudes y transacciones.

Consulta propia

Permite la consulta (no modificación) de peticiones, solicitudes y transacciones asociadas a los mismos back offices y organismos que están asignados al usuario que posee este permiso. Este permiso no permite la consulta de peticiones, solicitudes y transacciones ajenas al usuario (mediante filtro implícito en listados, prohibición de acceso a detalles, etc.).

Consulta de traza

Permite la consulta de las trazas generadas en el sistema.

Consulta propia de traza

Permite la consulta de las trazas generadas en el sistema sólo para aquellas relacionadas con el back office y organismo asociados al poseedor de este permiso. Cualquier otra traza será filtrada en los listados y no permite el acceso a cualquier detalle de ésta.

Consulta de usuarios

Permite consultar información de los usuarios del sistema.

Consulta propia de usuario

Permite a un usuario consultar sus propia configuración de usuario en el sistema. No permite el acceso a datos de cualquier otro usuario.

Modificación de usuario

Permite la modificación de los datos de usuarios.

Modificación propia de usuario

Permite la modificación de los datos del usuario poseedor de este rol. No permite la modificación de ningún otro usuario.
